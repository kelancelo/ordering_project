This app has 3 pages; Index, Selected Products and Checkout Done.
1) Index Page - displays all of the products. The user can add a product to their order by clicking 
the "Add to order" button of a product card and after clicking the button, a modal will show up
asking the user for the quantity and the add-ons they want to add. 

2) Selected Products Page - This page acts like a "shopping cart". The users can view the products
they selected and have the option to checkout.

3) Checkout Done Page - Shows an order summary similar to Selected Products Page with additional
information such as the order id, queue number, kiosk used to order and date.
